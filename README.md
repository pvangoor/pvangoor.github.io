# pvangoor.github.io
My github page

Run the site using `bundle exec jekyll serve`

## Jekkyl setup

Make sure you have the requirements: <http://jekyllrb.com/docs/installation/#requirements>

If you have not set up a ruby environment for your user, follow these steps:

```sh
mkdir ~/.ruby
echo 'export GEM_HOME=~/.ruby/' >> ~/.bashrc
echo 'export PATH="$PATH:~/.ruby/bin"' >> ~/.bashrc
source ~/.bashrc
```

Then install jekyll via:

`gem install bundler jekyll`

Install missing gems using

`bundle install`
